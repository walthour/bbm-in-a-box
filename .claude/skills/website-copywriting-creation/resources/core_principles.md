# Core Landing Page Principles

## Structural Flow (Cold Traffic)
1. Hook
2. Establish relevance
3. Prove
4. De-risk
5. Direct

## Non-Negotiables
- **Single primary CTA**: Repeated throughout, matching ad promise
- **Message match**: Hero headline/imagery mirrors ad hook and keyword intent
- **Social proof**: Appears early and often, not just at bottom
- **Visual hierarchy**: One dominant hero, modular sections, skimmable layout
- **Mobile-first**: Legible on all devices

## When to Use Long vs Short
- **Long form**: High complexity, price, or skepticism; cold traffic; requires education
- **Short form**: Simple offer, low friction, familiar audience; warm traffic

## Design for Scan-ability (CRITICAL)
> **The "Grunt Test"**: A user should understand the offer in 5 seconds without reading a single paragraph.

### 1. Visual Engineering
- **Encapsulation**: Use boxes, backgrounds, or distinct visual containers for key concepts. Don't just list things.
- **Whitespace**: Active element to force the eye to the next component. Surround CTAs with generous whitespace.
- **Visual Cues**: Use explicit cues (arrows, lines) and implicit cues (eye gaze) to direct attention.
- **Icons as Shorthand**: Use icons to anchor concepts. They process faster than text.

### 2. Writing for Scanners (The F-Pattern)
- **Front-Load Value**: Users read the first 2 words of a line. Bullet points MUST start with the benefit/action.
- **One Idea Per Block**: A visual "block" of text should never have more than one distinct idea.
- **"So What?" Test**: Every sentence must justify its existence. If it doesn't solve a problem, delete it.
- **Progressive Disclosure**: Hide secondary details. Use "Read More" or accordions for deeper info.

### 3. Key Frameworks
- **Z-Pattern (Simple)**: For Hero (Logo -> Nav -> Headline -> CTA).
- **F-Pattern (Content)**: For body. Critical content on left margin.
- **Gestalt Principles**: Group related items visually so the brain processes them as one unit.

### 4. Alex Hormozi Headline Formulas ($100M Offers)
- **The "How To" w/ Reversal**: How To [Dream Outcome] Without [Effort/Sacrifice].
- **The "Even If"**: [Dream Outcome] In [Timeframe] Even If [Common Obstacle].
- **The Value Equation**: Increase Perceived Likelihood + Decrease Time Delay & Effort.
- **Rule**: Be specific. "Lose 20lbs in 6 weeks" > "Get fit fast".

## Quick Checklist
- [ ] **Grunt Test Passed**: Offer clear in 5 seconds without reading body text?
- [ ] **Front-Loaded Bullets**: Do bullets start with the benefit?
- [ ] **Visual Anchors**: Are distinct sections encapsulated or anchored with icons/images?
- [ ] **No Text Walls**: Are paragraphs max 2-3 lines?
- [ ] **Flow**: Problem -> Solution -> Proof -> Offer -> Risk Reversal -> CTA
- [ ] **Proof**: Social proof in 2+ sections?
- [ ] **Message Match**: Does Hero match ad hook?
