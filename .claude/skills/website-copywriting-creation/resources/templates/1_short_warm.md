# Template 1: Short Warm Offer Page

## Best For
- Email subscribers or warm retargeting
- LinkedIn profile clicks
- Audience already knows you
- Low-friction offers

## Structure

### 1. Hero Strip
- **Headline**: Restate the offer clearly
- **Subheadline**: Outcome + mechanism (one line)
- **Primary CTA**: Clear button

### 2. Benefits Snapshot
- 3-4 benefit bullets or tiles with icons
- Optional micro-proof: "Used by 2,300+ [role]"

### 3. Visual of Outcome
- Product image or "after" state screenshot
- Optional short caption

### 4. Objection Mini-FAQ
- 3-5 Q&A addressing common concerns
  - "How long does it take?"
  - "Is this for me?"
  - "What if I'm not technical?"

### 5. Final CTA Strip
- Restate core promise in one line
- Same primary CTA
- Optional urgency or bonus mention

## Tone
Feel like a "confirmation nudge," not a full sales letter.
