# Template 3: Long-Form Sales Page

## Best For
- Higher price/commitment offers
- Complex products/services requiring trust
- High-ticket, subscriptions, medical, B2B
- Cold traffic needing education

## Macro-Structure Flow
Problem → Agitation → Dream State → Solution → Benefits → Proof → Offer → Objections → Close

## Structure

### 1. Hero Story Hook
- **Headline**: Outcome-driven + "who it's for" subhead
- Primary CTA (book call, start trial, purchase)
- Supporting visual (founder, product, before/after)

### 2. Problem & Agitation
- 2-3 short sections describing current pain
- Concrete scenarios the reader recognizes
- Transition: "What if you could…"

### 3. Solution Overview
- Introduce product/service
- 3-5 key pillars
- Visual diagram or step-by-step graphic

### 4. Deep Benefit Stack
- Each section = 1 big benefit
- Format: headline, short story/example, supporting visual
- Optional micro-case studies or quotes inside each

### 5. Social Proof Wall
- Testimonials, logos, case snippets
- Before/after metrics
- Optionally grouped by segment ("For agencies," "For clinicians")

### 6. Offer Breakdown
- Line-by-line "what you get" list (modules, features, services)
- Pricing and payment options
- Guarantee clearly stated

### 7. Risk Reversal & FAQ
- Guarantee, refund policy, compliance notes
- 6-10 FAQs targeting major objections
- Logistics: timing, access, support

### 8. Final CTA + Reiteration
- Simple recap: who this is for, what happens next
- Strong, singular CTA
- Optional urgency (legitimate only)

## Design Notes
- Treat each scroll as a "chapter" with its own job
- Multiple CTAs after persuasion peaks
- Frequent subheads (every 2-3 paragraphs)
- Mix formats to break text walls
