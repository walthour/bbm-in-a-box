# Websites

Web builds, landing pages, and research projects for your own brand (not client work — that lives in `active/clients/{slug}/active/websites/`).

Each website folder is self-contained. Common shape:

```
active/websites/{site-slug}/
├── README.md         # what this is, where it deploys
├── index.html        # or src/, app/, etc. depending on stack
├── .env.example
└── ...
```
